# Tableau-Dashboard-Pizza-Hut
Pizza Hut Sales Dashboard (Tableau)
📌 Project Overview
This project presents an interactive Tableau dashboard developed using Pizza Hut’s sales data to uncover meaningful business insights. The primary goal of the dashboard is to provide a comprehensive view of sales performance, customer purchasing patterns, and product popularity.
By leveraging Tableau’s powerful visualization capabilities, the dashboard transforms raw data into easy-to-understand charts and KPIs that can support business leaders and managers in making informed, data-driven decisions.

🎯 Key Objectives
Analyze sales performance across different product categories.
Identify top-selling pizzas and their contribution to overall revenue.
Track sales trends over time (daily, weekly, monthly).
Understand order patterns and customer preferences.
Provide an interactive and user-friendly dashboard for quick decision-making.

📊 Features of the Dashboard
Sales Overview → Total sales, revenue trends, and KPIs at a glance.
Top-Selling Products → Bar chart ranking pizzas by sales volume and revenue.
Sales by Category → Insights into classic, supreme, veggie, and other categories.
Time-Based Trends → Line charts to monitor daily/monthly sales growth.
Interactive Filters → Select category, time range, or product type for deeper analysis.

🛠️ Tools & Technologies
Tableau → For data visualization and interactive dashboards.
Excel/CSV Dataset → Pre-processed Pizza Hut sales data.

🚀 Insights
Identified the most profitable pizza types.
Observed seasonal sales peaks during weekends and holidays.
Found that specific categories drive the majority of revenue.
